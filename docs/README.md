# 💎 GlanceCore

**GlanceCore** — is a modular widget ecosystem for Windows, built on **.NET 10** and **WPF**. The project seamlessly blends Apple’s design philosophy (minimalism, continuous corners) with powerful **Liquid Glass** visualization technologies.

## ✨ Features

*   **Liquid Glass Design:** Hardware-accelerated blur (Mica/Acrylic) combined with custom HLSL shader refractions.
*   **Intuitive Hub:** A unified control center for managing widgets, global settings, and the plugin store.
*   **Modular Architecture:** Every widget operates as an independent, isolated module.
*   **Total Customization:** Freedom to change fonts, sizes, transparency, and scale of any element to fit your desktop perfectly.
*   **High Performance:** Native 60+ FPS animations with minimal resource footprint thanks to .NET 10 optimizations.

## 🛠 Tech Stack

*   **Platform:** .NET 10 (WPF)
*   **Language:** C# 14
*   **Design Language:** Fluent Design / Apple-style Minimal
*   **Visual Effects:** DWM API (Mica), HLSL Shaders (Refraction)
*   **Architecture:** Plugin-based (Interface-driven)

## 📂 Project Structure

*   **`/Core`** — System Kernel (Auto-start, Configuration, Widget Host).
*   **`/UI`** — Shared resources, styles, dictionaries, and shaders.
*   **`/Widgets`** — A collection of standard modules (Hardware, Weather, Finance, Time).
*   **`/Plugins`** — SDK for third-party extension development.
*   **`/Views`** — Main Hub interface and control panels.
